What is ChatCloud?

ChatCloud is a very simple web application that allows you to create wordclouds from Facebook chats. The idea was born after seeing Schmavery's facebook-chat-api, a Node.js module that emulates browser actions to access Facebook chats. After seeing such a useful module, I wondered what would be the best way to utilize it. As I couldn't think of anything useful, I decided making a wordcloud was the most logical thing to do. The wordclouds are generated with the help of two of timdream's projects, wordfreq and wordcloud2.js.

Why is there no official Facebook login method?

The original Facebook Chat API was deprecated and taken out of service, as what is done to all useful API's eventually. The current method with Schnavery's module requires login credentials so it can emulate the browser actions necessary to reach the messages page and extract the information. Your login information is not stored by this app, and isn't used for any other purpose than the wordclouds. However, I can't guarantee that some malicious third party will find an exploit, so use this app at your own risk.
